🔴 Server B (port 4000)

This is a different origin. Go to http://localhost:3000 to run the experiments.

📡 This server exists to demonstrate SOP

Server B runs on http://localhost:4000 — a different origin from Server A (http://localhost:3000).

The experiments on Server A will try to access this server's data, DOM, and cookies. The browser will block most of these attempts because of the Same-Origin Policy.

Go to http://localhost:3000 to start the experiments.

🔗 Server B Endpoints

GET /api/secret-data — Returns secret JSON data
GET /set-cookie — Sets a cookie on this origin
GET /check-cookies — Shows all cookies visible on this origin
GET /iframe-page — A page loaded in iframe on Server A
GET /postmessage-page — A page that listens for postMessage
