🔐 Login

This form sends a POST request. Watch the Network tab!

🧠 What to Observe

1. Open DevTools → Network tab before clicking Login.
2. Submit the form and notice the POST request (not GET!).
3. Click on the request → look at Payload — your username and password are sent in the request body.
4. Notice: on plain HTTP, this data is sent in cleartext. That's why HTTPS matters (Project 12)!
