🔐 Login

This form sends a POST request. Watch the Network tab!

🧠 What to Observe

1. Open DevTools → Network tab before clicking Login.
2. Submit the form — the page does NOT reload! The POST request stays in the Network tab.
3. Click the POST /login request → Headers tab shows request/response headers.
4. Click Payload tab — your username and password are in the request body in cleartext.
5. Click Response tab — see the HTML the server sent back.
6. Also check your terminal — the server logged everything.
7. 💡 Tip: also check "Preserve log" in Network tab to keep logs across navigations.
