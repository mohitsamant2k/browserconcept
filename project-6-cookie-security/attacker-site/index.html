📊 SameSite Behavior Summary

Request Type SameSite=None SameSite=Lax SameSite=Strict
Top-level link (GET) Sent Sent ✅ Blocked
Form POST Sent Blocked Blocked
fetch() / XHR Sent Blocked Blocked
<img> / <script> Sent Blocked Blocked
iframe Sent Blocked Blocked
window.open (GET) Sent Sent ✅ Blocked

💡 Lax is the browser default — it blocks most cross-site attacks while still letting users follow links naturally.
💡 Strict blocks everything — safer but can break "Login with Google" flows.
