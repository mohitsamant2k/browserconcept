🎯 Clickjacking Variations

1. Classic Clickjacking (this demo)

Invisible iframe + fake button overlay. User thinks they click one thing, actually clicks another.

2. Likejacking

Hidden Facebook "Like" button under a fake button. User unknowingly likes an attacker's page.

3. Cursorjacking

Custom CSS cursor offset from the real cursor position. User clicks in the wrong place.

4. Multi-step Clickjacking

Tricking user through a series of clicks — e.g., a "game" where each click changes their settings.

5. Drag-and-Drop Clickjacking

User drags an element in a "game" but actually drags data across iframe boundaries.
